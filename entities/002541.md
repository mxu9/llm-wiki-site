# 宏桥控股（002541）

| 项目 | 内容 |
|------|------|
| **定位** | 全球最大电解铝生产商之一，中国宏桥(01378.HK)核心运营资产 |
| **上市方式** | 2025年12月借壳上市(A股)，中国宏桥持股**89%** |
| **产业链** | 铝土矿→氧化铝→电解铝→铝加工→再生铝，**全产业链一体化** |
| **产能** | 电解铝**646万吨**、氧化铝**2,100万吨**（均全球前列） |
| **铝土矿** | 几内亚权益产能1,125万吨，供应基地约6,000万吨/年，自给率>100% |
| **电力** | 山东自备电厂(自给率46%) + 云南绿色水电 |
