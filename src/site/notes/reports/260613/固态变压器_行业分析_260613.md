## Step 1: 研究范围界定

| 维度 | 定义 |
|------|------|
| **行业** | 固态变压器（Solid-State Transformer, SST） |
| **技术路线** | 基于SiC/GaN功率半导体的高频电力电子变换，实现中压交流→低压直流一步转换 |
| **核心场景** | AIDC数据中心（首要）、充电桩、新能源并网、轨道交通、微电网 |
| **研究目的** | 行业格局研判 + 产业化进程评估 + A股投资标的梳理 |
| **深度** | 全景概览（10-15页级） |
| **样本** | A股15家+海外3家核心厂商 |
| **产业链** | 上游SiC器件/高频变压器/磁芯 → 中游SST整机 → 下游AIDC/电网/充电桩 |

### 2.2 技术原理与颠覆性

| 对比维度 | 传统工频变压器 | 固态变压器(SST) |
|----------|--------------|----------------|
| 变换步骤 | 中压AC→降压→UPS整流→电池→逆变→服务器整流（6步） | **中压AC→低压DC（1步）** |
| 整机效率 | 90%-92% | **98.5%+** |
| 体积 | 基准 | **减少60%+** |
| 电力损耗 | 基准 | **降低40%+** |
| 铜材用量 | 基准 | **减少45%** |
| 单机功率密度 | <200kW | **>600kW** |
| 成本 | 基准 | 3-5倍（当前） |
| 智能化 | 无 | 支持功率因数校正/电能质量治理/交直流混合 |

> 🔑 **核心类比**（华福证券）：**SST之于电力系统，正如CPO之于光通信。** 均实现高度集成+减少功耗。

### 2.3 产业化进程

| 阶段 | 时间 | 特征 |
|------|------|------|

| **储备期** 🔥 | **2025-2026** | 密集发布新品，华为战略定调，TOP10云厂商导入 |
