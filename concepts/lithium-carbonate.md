# 碳酸锂板块

>
> 🔥 本周核心：宜春枧下窝采矿项目核准（年产能10→3万吨LCE），最大利空落地；锂电排产6月环比+5%、7月环比+5%/同比+50-90%，Q3旺季+估值切换行情。

## 7/1 关键事件

| 事件 | 详情 |
|------|------|
| **永兴材料季报期推荐** | Q2单季预计6亿，扣纯现金后年化9x；季度业绩锂矿板块最容易预测、兑现度最好 |
| **宁德时代：美国逆变器禁令影响极有限** | 中国逆变器公司产品本质安全有保障，不改长期趋势 |

## 6/29 关键事件

| 事件 | 详情 |
|------|------|
| **宜春枧下窝采矿核准** | 年产能由10万吨缩至3万吨LCE，市场视为供应端最大利空落地 |
| **锂电排产** | 6月环比+5%，7月环比+5%/同比+50-90%，Q3旺季+估值切换 |
| **欧盟储能协议** | 2026-2028年45GW目标，2030年累计200GW，储能需求持续拉动了锂电产业链 |
