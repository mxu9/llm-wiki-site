# 电解液

> 🔥 核心逻辑：六氟磷酸锂25Q3触底反弹(+288%) → 26年紧平衡 → Q4缺口 | VC需求+47% | 7月中报业绩暴增

## 概览

锂离子电池四大主材之一，由溶质（六氟磷酸锂LiPF₆ / LiFSI）+ 溶剂（碳酸酯）+ 添加剂（VC/FEC）构成。中国出货全球占比94%，2025年208万吨(+42%)。

## 产业链

```
碳酸锂/萤石 → 六氟磷酸锂(LiPF₆) ─┐
                                  ├→ 电解液 → 锂电池
碳酸酯(DMC/EMC/EC) ──────────────┤
VC/FEC添加剂 ─────────────────────┘
LiFSI新型锂盐（快充/800V增量）────┘
```
