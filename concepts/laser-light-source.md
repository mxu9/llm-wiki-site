# 激光光源

> AI光模块/光通信核心上游：半导体激光芯片 + 光纤激光器 + CPO外置光源 + 激光加工设备

## 产业链结构

```
半导体激光芯片（长光华芯/源杰科技）
    ├── 光纤激光器（锐科激光/杰普特/英诺激光）
    │       ├── 光调制器/隔离器（光库科技）
    │       ├── CPO外置光源 → AI光互联（杰普特/源杰/仕佳光子）
    │       └── 激光加工设备（大族激光/华工科技/帝尔激光）
    └── 光学晶体/非线性晶体 → 固体激光器
```
