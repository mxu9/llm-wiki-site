# GW星座（中国星网）

> 中国首个巨型卫星互联网计划，首个空天一体6G互联网计划。国资委直管，中国卫星网络集团运营，规划12,992颗卫星，国家战略级工程。

## 核心数据

| 维度 | 详情 |
|------|------|
| **运营方** | 中国卫星网络集团有限公司（国资委100%） |
| **总规划** | **12,992颗** |
| **双层架构** | GW-A59（6080颗/500km极低轨）+ GW-A2（6912颗/1145km近地轨） |
| **当前在轨** | **171颗**（2026年6月） |
| **2026计划** | 发射300-700颗，年底建成骨干网；**启动大规模招标（意向→实质合同）** |
| **近期目标** | 2029年前完成~10%部署（~1300颗） |
| **终态目标** | 2035年全部完成 |
| **定位** | 战略通信安全 + 全域宽带覆盖 + 6G空天地一体 + 手机直连 |
