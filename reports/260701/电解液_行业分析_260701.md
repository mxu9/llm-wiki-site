## Step 1 — Define Scope

**电解液** = 锂离子电池四大主材之一，由溶质（六氟磷酸锂LiPF₆/LiFSI）+ 溶剂（碳酸酯类）+ 添加剂（VC/FEC等）构成。

- **子环节**：六氟磷酸锂（核心溶质，成本占比~50%）、LiFSI新型锂盐、VC/FEC添加剂、溶剂
- **覆盖标的**：天赐材料(002709) / 新宙邦(300037) / 多氟多(002407) / 石大胜华(603026) / 永太科技(002326) / 天际股份(002759) / 瑞泰新材(301238) / 华盛锂电(688353) + 宏源药业

### 供需格局

| 维度 | 判断 |
|------|------|
| **六氟** | 25Q3触底反弹，25Q4-26H1紧平衡。中信建投：2026全年六氟总需求~40.4万吨 vs 有效产能~41.4万吨，**Q4面临缺口** |

| **LiFSI** | 添加比例从2%(2025)→4%(2030E)，2030年需求30万吨。当前价~8.5万/吨 |
| **锁单潮** | 天赐/新宙邦/永太等累计获长单超400万吨，锁的是上游原料而非电解液本身 |

### 价格趋势

六氟磷酸锂价格走势：25年7月低点4.9万 → 25年底17万（+230%）→ 26年5月短暂回升至11.4万 → 6月小幅回落至10.63-10.80万。**中信建投：Q2中下旬加工费上涨，5月百川报价已涨2000元。天赐材料：行业库存仅1-2周，产能利用率将达80%，涨价基础坚实。**
